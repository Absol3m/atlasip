# Changelog — 0.2.x Series

## 0.2.4 — Early Enrichment Pipeline
### Added
- First implementation of the enrichment workflow
- Reverse DNS lookup
- Basic WHOIS extraction
- ASN parsing logic
- Loading states in the UI

### Changed
- Improved error handling for network failures
- Refactored backend modules for clarity

### Fixed
- Fixed blocking behavior during long lookups
- Corrected IPv6 parsing inconsistencies

### Added
- First iteration of a structured results panel

- Basic theming (light/dark)

### Changed
- Updated typography and spacing
- Improved input validation feedback

### Fixed
- Fixed layout issues on small window sizes
