# Changelog — 0.3.0

## Overview
Version 0.3.0 is a major milestone focused on UI/UX modernization, internal consistency, configuration improvements, and documentation restructuring. This release introduces design tokens, a unified settings panel, a fully reorganized documentation system, and a clean, professional user experience.

---

## Added
### UI/UX
- Complete UI redesign using a modern design system
- Introduction of design tokens for colors, spacing, typography, and components
- Unified Settings panel with:
  - Network settings
  - UI preferences
  - Logging configuration
- New responsive layout for the main analysis page
- Improved loading indicators and error states

### Configuration
- Strongly typed configuration schema
- Automatic validation and default value restoration
- New logging configuration section
- Proxy configuration support (address + port)

### Documentation
- Full rewrite of the User Manual:
  - Getting Started
  - Installation
  - Configuration
  - UI Overview
  - Troubleshooting
- New Developer documentation:
  - Build instructions
  - Release process
  - Configuration schema
- Clean separation between public docs and internal notes

### Release Engineering
- Standardized artifact structure for macOS and Windows
- Verified DMG and MSI outputs
- Improved reproducibility of builds
- Cleaned and validated workspace before release

---

## Changed
### UI
- Harmonized spacing, typography, and component behavior
- Replaced hardcoded values with design tokens
- Updated results panel for clarity and readability

### Backend
- Refactored enrichment pipeline for stability
- Improved error propagation and logging
- Updated configuration loader for schema evolution

### Documentation
- Removed legacy TXT documentation
- Introduced Markdown‑based structured documentation
- Reorganized `docs/` directory for clarity

---

## Fixed
- Fixed inconsistent behavior in timeout handling
- Corrected proxy mode edge cases
- Fixed UI state desynchronization after long operations
- Resolved issues with configuration not persisting correctly
- Fixed Windows installer metadata inconsistencies

---

## Removed
- Deprecated TXT documentation files
- Legacy UI components replaced by the new design system
- Old configuration fields no longer used

---

## Notes
This version establishes the foundation for future features, including:
- Advanced enrichment modules
- Local geolocation database integration
- Plugin system (planned)
- Automated update mechanism (planned)
